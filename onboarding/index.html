Install

Add the app to your home screen

The app works best when installed from Safari on iPhone so it launches like a normal app instead of a browser tab.

1
Open the app in Safari
Open the site link in Safari, not another browser. On iPhone, Safari gives the cleanest install path for a PWA.
2
Tap the Share button
The Share button is the square with the upward arrow. Scroll the share sheet until you see the install option.
3
Choose “Add to Home Screen”
Confirm the name, then tap Add. The app icon will appear on your home screen like a normal installed app.
4
Open it from the new icon
Launch from the home screen icon after installing. That gives the clean app-style view with less browser clutter.

Use

How to use the stationing screen

The workflow is simple: pick the correct project, wait for good GPS, then read station, side, and offset.

Pick the right project
Use the project dropdown first. If the wrong centerline file is selected, the stationing result will be wrong even if your GPS is good.
Wait for the GPS to settle
Give the phone a few seconds while standing still. The app smooths multiple GPS points together to reduce random jumpiness.
Read the result row
Station shows position along the centerline. Side shows LT or RT. Offset shows how far you are from the line.
Fast mode
Fast mode reacts quicker and uses fewer smoothing points. It feels more responsive, but it can move around more.
Slow mode
Slow mode uses tighter GPS filtering and more smoothing points. It updates slower, but the result is usually steadier.
Save Points

Store field notes locally on the device

Saved points are meant for quick field reference. They keep the station, side, offset, note, time saved, and coordinates.

1
Wait for a live station result
You cannot save a meaningful point until the app has a current GPS lock and a displayed stationing result.
2
Type a note
Add whatever matters: issue location, punch item, utility conflict, picture reference, or inspector note.
3
Tap Save Point
The point is stored on that device and listed in the Saved Points section below the live result card.
4
Reopen later and review
Saved points should still be there when you reopen the app on the same device and browser context.
Saved points are local to the device. They are not a cloud database. If device storage is cleared, private browsing is used, or the browser wipes site data, those saved points can disappear.
Capture

Save an image of the current stationing card

The app uses a manual canvas capture instead of Safari DOM screenshot tricks. That makes it more stable inside iPhone PWAs.

What gets captured
The capture image includes the visible station, side, offset, project, smoothing count, status text, and the note currently typed into the note box.
How to save on iPhone
After tapping Capture, press and hold the preview image and save it to Photos, or use the provided save/open flow depending on the build.
Troubleshooting

What to check when something looks off

Most bad results come from bad GPS conditions, the wrong project file, or moving too quickly while trying to read a value.

Station seems wrong
Confirm the correct project is selected. Then stop moving and let the GPS settle. If needed, switch to slow mode for a steadier result.
Offset is jumping around
That usually means poor GPS accuracy, tree cover, nearby structures, or that you are still moving while the phone is trying to smooth points.
Saved points disappeared
Check whether the app was opened in the same browser/app context. Private browsing and aggressive storage cleanup can wipe local saved data.
Capture will not save
On iPhone, use the preview flow and save the generated image from there. Desktop and some Android browsers can usually download directly.
