What This App Does

Uses GPS and a project centerline to give you real-time station, side (LT/RT), and offset in the field.

Designed for inspectors, layout checks, and quick station reference.

Add to Home Screen

iPhone:

  1. Open in Safari
  2. Tap the Share button
  3. Select Add to Home Screen

Android:

  1. Open in Chrome
  2. Tap menu (3 dots)
  3. Select Add to Home screen

How to Use

  1. Select the correct project
  2. Allow GPS access
  3. Wait for accuracy to stabilize
  4. Read station / side / offset
  5. Add a note if needed
  6. Use capture button if needed
